#### docz 文档 的使用文档

	一个非常有趣的基于markdown，支持jsx组件的文档工具

##### feature

- 0配置

- 非常快-完全支持webpack4和自动代码切分

- 自定义方便

- 基于MDX 组件化markdown

- 插件化

- 支持typescript

##### 使用方法

	1. 在windows环境下， 需要升级nodejs到新版本（8.11可以满足需求）
	2. 需要切换到管理员状态的命令行工具运行后续指令

1. 初始化

```
yarn add docz --dev
```

2. 运行文档

```
docz dev
```
